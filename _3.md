# 전기기능사 학습 플랫폼 "전기마스터" 서비스 설계서

## 1. 서비스 개요
- **서비스명:** 전기마스터 (ElecMaster)
- **목표:** 전기기능사 필기 및 실기 시험 준비생들을 위한 올인원 학습 솔루션 제공
- **타겟 사용자:** 전기기능사 자격증 취득을 준비하는 수험생

## 2. 주요 기능 및 화면 구성

### 1) 메인 페이지 (Landing Page)
- 서비스 소개 및 핵심 기능 요약
- 실시간 시험 일정 카운트다운
- 주요 학습 메뉴 바로가기 (핵심요약, 기출문제, AI 챗봇)
- 합격 수기 및 학습 통계

### 2) 핵심 요약 (Study Notes)
- 과목별(전기이론, 전기기기, 전기설비) 핵심 개념 정리
- 중요도에 따른 별표 표시 및 검색 기능
- 이미지 및 도표를 활용한 시각적 설명

### 3) 기출문제 (CBT Practice)
- 연도별/회차별 기출문제 풀이 모드
- 실제 시험과 유사한 CBT 환경 구현
- 오답 노트 기능 및 성적 분석 그래프

### 4) AI 학습 챗봇 (AI Tutor)
- OpenAI API 등을 활용한 실시간 질의응답
- 복잡한 공식이나 이론에 대한 쉬운 설명 제공
- 학습 스케줄 관리 및 맞춤형 문제 추천

## 3. 기술 스택 (제안)
- **Frontend:** React.js / Tailwind CSS (빠르고 반응성이 뛰어난 UI)
- **Backend:** Node.js / Python (FastAPI)
- **AI Integration:** OpenAI GPT-4 API
- **Database:** PostgreSQL (사용자 데이터 및 문제 은행 관리)

## 4. 디자인 컨셉
- **메인 컬러:** 신뢰를 주는 Deep Blue (#1E3A8A) 및 전문적인 Teal (#0D9488)
- **폰트:** 가독성이 높은 Sans-serif (Pretendard 등)
- **레이아웃:** 학습에 집중할 수 있는 깔끔한 대시보드 형태의 레이아웃
